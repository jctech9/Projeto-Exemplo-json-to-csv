
1. INICIAR O SERVIDOR SPRING BOOT - Windows

Comando:
  .\mvnw spring-boot:run


2. TESTE COM ARQUIVO JSON LOCAL

Comando:
  curl.exe -X POST http://localhost:8080/export/xlsx `
    -H "Content-Type: application/json" `
    -d "@formatoPadrao.json" `
    -o dados.xlsx

Resultado:
  - Arquivo "dados.xlsx" gerado na pasta demo


3. TESTE COM JSON DE API EXTERNA

Passo 1 - Buscar dados da API:
  curl.exe -X GET "http://localhost:8090/processos" `
    -H "accept: application/json" `
    -o temp.json

Passo 2 - Converter para Excel:
  curl.exe -X POST http://localhost:8080/export/xlsx `
    -H "Content-Type: application/json; charset=utf-8" `
    -d "@temp.json" `
    -o dados.xlsx


4. FORMATO JSON ACEITO - ExcelService.java

O endpoint aceita um objeto JSON onde:
  - Cada chave = nome da aba no Excel
  - Cada valor = array de objetos (linhas da aba)
  - Cada objeto = conjunto chave-valor (colunas)


curl.exe -X POST http://localhost:8080/export/xlsx/auto -o dados.xlsx
