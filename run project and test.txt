
ANTES DE EXECUTAR O PROJETO, É PRECISO ESTÁ LOGADO NO SISTEMA GESTÃO DE RISCOS

OS COMANDOS ABAIXO SÃO PARA WINDOWS 

1. INICIAR SERVIDOR NO WINDOWS- dentro da pasta do projeto demo execute: no powwershell
   .\mvnw spring-boot:run

2. EXPORTAR JSON REMOTO (API)
   
  2.1. EXPORTAR JSON REMOTO (API) PASSANDO ID
    curl -X POST "http://localhost:8080/export/xlsx/api" -H "Content-Type: application/json" -d "{\"id\": 63}" -o "saida.xlsx"
