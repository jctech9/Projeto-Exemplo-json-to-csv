

1. INICIAR SERVIDOR NO WINDOWS
   .\mvnw spring-boot:run

2. EXPORTAR JSON LOCAL
   curl.exe -X POST http://localhost:8080/export/xlsx ^
     -H "Content-Type: application/json" ^
     -d "@json/01-processo-controller.json" ^
     -o saida.xlsx

3. EXPORTAR JSON REMOTO (API)
   curl.exe -X POST http://localhost:8080/export/xlsx/auto -o saida.xlsx

4. FORMATO JSON
   {
     "NomeAba1": [
       {"coluna1": "valor1", "coluna2": "valor2"},
       {"coluna1": "valor3", "coluna2": "valor4"}
     ],
     "NomeAba2": [...]
   }

ENDPOINTS:
  POST /export/xlsx      - JSON no body
  POST /export/xlsx/auto - JSON de API externa

LOCALIZAÇÃO:
  Projeto: demo/
  JSONs exemplo: demo/json/
